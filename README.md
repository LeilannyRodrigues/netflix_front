# 🎬 Netflix Clone - Imersão Alura

Repositório destinado ao projeto de uma réplica da interface da Netflix, desenvolvido durante a **Imersão Alura**. O projeto foca em Front-end responsivo e interatividade com trailers.

## 📌 Sobre o Projeto
O objetivo deste projeto foi recriar a experiência de usuário da plataforma Netflix, incluindo a tela de seleção de perfis e o catálogo principal com funcionalidades de *hover* para visualização de trailers integrados diretamente do YouTube.

- **Acesse agora:** [https://netflix-eight-liart.vercel.app/index.html]

## 🚀 Funcionalidades
* **Seleção de Perfis:** Interface inicial interativa para escolha de usuário.
* **Catálogo Dinâmico:** Listagem de filmes e séries divididos por categorias (Mais Assistidos, Séries, Para Maratonar).
* **Player Integrado:** Ao passar o mouse ou clicar em um título, um modal/container exibe o trailer oficial via API do YouTube.
* **Responsividade:** Interface adaptável para diferentes tamanhos de tela.
* **Menu de Navegação:** Header fixo com links funcionais para seções da página.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica da página.
* **CSS3:** Estilização avançada, uso de Flexbox, Grid e animações de transição.
* **JavaScript:** Lógica para alternância de perfis e manipulação do player de vídeo.
* **YouTube API/Embed:** Utilizado para carregar os trailers dinamicamente.

## 📂 Estrutura de Arquivos
* `index.html`: Tela de seleção de perfis (Quem está assistindo?).
* `catalogo.html`: Página principal com a vitrine de conteúdos.
* `css/`: Pasta contendo os arquivos de estilização e reset.
* `js/`: Scripts para interatividade da galeria e transições.

## 🎨 Demonstração
O layout segue fielmente a identidade visual da Netflix, utilizando a paleta de cores característica (Preto, Vermelho e Cinza) e a tipografia padrão.

> **Nota:** Este projeto possui fins estritamente educacionais para estudo de desenvolvimento web front-end.
